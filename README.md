leopard-cli

# Usage
<!-- usage -->
```sh-session
$ npm install -g leopard-cli
$ leopard COMMAND
running command...
$ leopard (-v|--version|version)
leopard-cli/0.0.0 win32-x64 node-v10.6.0
$ leopard --help [COMMAND]
USAGE
  $ leopard COMMAND
...
```
